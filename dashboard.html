CSV Import Instructions

Singles Format

For singles matches, your CSV should have the following columns:

  • Date (dd/mm/yy or dd/mm/yyyy)
  • Opponent Name
  • My Level (1-12)
  • Opponent Level (1-12)
  • Score (format: 21-19 21-15)
  • Extra Points (optional)
  • Blank column
  • Phase/Tournament (optional)
  • Duration (minutes, optional)
14/12/24;John Doe;12;11;21-19 21-15;0;;IC Tournament;45
Doubles/Mixed Format

For doubles and mixed matches, your CSV should have the following columns:

  • Date (dd/mm/yy or dd/mm/yyyy)
  • Teammate Name
  • Discipline (D for Doubles, M for Mixed)
  • My Level (1-12)
  • Teammate Level (1-12)
  • Opponent 1 Name
  • Opponent 1 Level (1-12)
  • Opponent 2 Name (optional, leave empty for 2v1)
  • Opponent 2 Level (optional, leave empty for 2v1)
  • Score (format: 21-19 21-15)
  • Phase/Tournament (optional)
  • Duration (minutes, optional)
14/12/24;Jane Smith;D;12;11;John Doe;11;Mike Johnson;12;21-19 21-15;Club Tournament;60

Note: Columns must be separated by semicolons (;). The first row should contain headers.

When exporting, the system automatically includes scraped time and extra type information in additional columns.
